# 🎓 StudyFlow — Ultimate Academic Workspace

> Organize your academic chaos with a premium, sleek, and highly secure digital workspace. Built specifically to streamline university routines, assignments, and essential web resources.

[![Live Demo](https://img.shields.io/badge/Demo-Visit%20Workspace-FF5722?style=for-the-badge&logo=vercel)](https://v-ruby.vercel.app/)

---

## ✨ Key Features

* **📅 Dynamic Class Routine:** Visually elegant schedule tracker with built-in **Ramadan Mode** custom timings.
* **📝 C&A Organizer:** Smart class test and assignment tracker with automatic urgency glowing indicators (Green = Safe, Orange = Soon, Red = Overdue).
* **🔗 Categorized Link Vault:** Save and star your frequent web links (Official, Materials, Tools, Community).
* **🔐 Hidden Secure Vault:** Store sensitive URLs behind a dedicated PIN/password barrier.
* **📢 Admin Broadcast Panel:** Role-based global notice board for immediate updates.
* **📱 Progressive Native Ready:** Fully optimized for mobile with customized safe-area paddings and Capacitor dynamic status bar integration.
